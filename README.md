# 🔐 Fullstack Login System

A Full Stack Login System built using React, Spring Boot, and MySQL. This project allows users to register, log in, update their profile, upload a profile picture, change their password, and reset forgotten passwords.

---

## 🚀 Live Demo

### Frontend
https://fullstack-login-system.vercel.app

### Backend
https://fullstack-login-system-production.up.railway.app

---

## ✨ Features

- User Registration
- User Login
- Profile Update
- Profile Image Upload
- Change Password
- Forgot Password
- Responsive User Interface
- MySQL Cloud Database
- Secure REST APIs

## ⚙️ How to Run

### Backend

```bash
cd backend/LoginProject
mvn spring-boot:run
```

### Frontend

```bash
cd frontend/login-frontend
npm install
npm run dev
```
